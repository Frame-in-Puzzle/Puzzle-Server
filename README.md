## Tech Stack

- Java Version : Java11
- Compiler : Gradle(2.5.6)
- IDE : IntellijIDEA
- Framework : Spring Boot
- DB : MariaDB
- Library : JPA(QueryDsl), Spring Security, Swagger, Lombok, Junit, JaCoCo, AWS Service.. etc
